Selos visuais das potências principais.

Arquivos esperados:

- `gob.png`
- `cmsb.png`
- `comab.png`

Também são aceitos `.jpg`, `.jpeg` e `.webp` com os mesmos nomes. PNG com fundo transparente é o ideal. Se a imagem tiver fundo branco ou xadrez muito claro, o renderizador tenta remover pixels quase brancos antes de aplicar o selo.

No cadastro, `potencia` deve ser apenas `GOB`, `CMSB` ou `COMAB`; o campo `potencia_complemento` guarda a jurisdição ou obediência complementar, por exemplo `GOB-RS`, `GLMERGS`, `GORGS`, `GOSC` ou `GOP`.
