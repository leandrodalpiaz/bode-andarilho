# Templates visuais

Coloque aqui o template padrão do sistema:

```text
assets/templates/default_event_card.png
```

Recomendação:

- formato: PNG
- proporção vertical
- largura ideal: 1080 px ou mais
- área central livre para texto
- evitar texto fixo no centro da arte

Enquanto uma Loja não tiver template próprio configurado, o renderizador usará esse arquivo como fallback visual.
