Fontes `.ttf` usadas no card visual padrão.

Arquivos versionados inicialmente:

- `Cinzel-Regular.ttf`
- `CormorantGaramond-SemiBold.ttf`

As licenças OFL estão nos arquivos `OFL-*.txt`.

O renderizador também reconhece `Cinzel-SemiBold.ttf` e `LibreBaskerville-Regular.ttf` caso sejam adicionadas depois. Se nenhuma fonte existir, o bot usa fontes serifadas disponíveis no sistema como fallback.
